# To-Do C++ Web Application

## Описание проекта

Данный проект представляет собой веб-приложение для управления задачами (To-Do List).

Пользователь может:

- добавлять задачи;

- просматривать список задач;

- удалять задачи;

- отмечать задачи как выполненные.

---

## Архитектура

Проект реализован по клиент-серверной архитектуре:

- **Frontend** — веб-интерфейс (HTML, CSS, JavaScript)

- **Backend** — сервер на C++
